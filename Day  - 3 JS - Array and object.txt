Array :

Group of elements as one entity
using  []
It allowed duplicates and heterogenous(dis similar) elements
Array elements are storing based on indexing (0 to n-1)
Negative (-ve) indexing is not possible
Array is a iterable object using(for loop , while loop , do while loop , for-of)
length property - to find size of array using (eid.length)


JavaScript Object :

Group of key:values ('eid' : 101) pairs as one entity
using {}
Duplicate keys are not allowed but values are allowed
indexing is not applicable
group of data & functionality as one entity
including concept not applicable
JS object is not iterable object but we can iterate using for-in(newly )
no length property
